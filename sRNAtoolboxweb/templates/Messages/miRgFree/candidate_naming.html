Unless specified otherwise, miRgFree will detect the closest species in MirGeneDB using the species detection functionality introduced in miRNAqc. microRNA sequences in MirGeneDB will be used to detect if the predicted novel microRNAs are known in other species or at least if they belong to a known microRNA family.

Names will be assigned as follows:

  • First, candidates that match perfectly a known microRNA sequence from any of the specified reference species will receive that name.
  • Second, perfect matches from other reference sequences in MirGeneDB will be named accordingly
  • If a sequence match is not found, a seed match will be attempted. All microRNAs from the whole database with the same seed sequence are considered, assigning in this order if a seed match is found:
    • Name of that miRNA in any of the reference species
    • The species with overall lowest number of mismatches between the candidate and mature sequences
